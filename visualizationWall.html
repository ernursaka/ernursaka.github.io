Interactive Registration Tool

Observing the details of a high resolution image by using small low resolution screens is difficult for humans. To make it easy, we decided to use high performance computing (HPC) Cluster and Visualization Wall.

In this project, our goal was to prepare a program for users to display their images by using different number of screens and giving an opportunity to display more than one image to see them together. To achieve these tasks, We implemented two programs with different technologies:.

ErImageShow: Shows one or more images by using at most 15 screens. Using python programming language, gThumb (gThumb, 2010) image viewer, and Xdmx

DivShow: Shows one image by using all screens. Using C programming language, OpenGL, Glut, libtiff (LibTIFF - TIFF Library and Utilities), and Chromium.
